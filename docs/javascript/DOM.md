### `Element.scrollIntoView()`

通过滚动元素的父容器使得内部元素可见。

``` js
el.scrollIntoView()
el.scrollIntoView({
  behavior: 'auto', // 'auto' | 'smooth'
  block: 'start', // 垂直方向的对齐 'start' | 'center' | 'end' | 'nearest'
  inline: 'nearest', // 水平方向的对齐 'start' | 'center' | 'end' | 'nearest'
})
```
